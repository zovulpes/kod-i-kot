# Код и Кот

**Интерактивный графический тренажёр для изучения процедурного программирования.**  
Приложение позволяет визуализировать выполнение алгоритмов, закреплять знания через практические задания и тесты. Главным символом тренажёра является кот, который сопровождает пользователя и мотивирует на обучение.

---

## Возможности
- Запуск в браузере без установки дополнительного ПО.  
- Начало нового обучения или продолжение сохранённого прогресса (LocalStorage).  
- Пошаговые задания с визуализацией алгоритмов.  
- Графическое отображение алгоритмов (блок-схемы).  
- Запуск и остановка выполнения заданий.  
- Выбор уровня сложности.  
- Подсказки и обратная связь о результатах.  
- Отслеживание прогресса пользователя.  
- Взаимодействие с котом-символом для подсказок.

---

## Ограничения
- Только HTML, CSS и JavaScript (без серверной части).  
- Лёгкий объём кода для работы на слабых компьютерах.  
- Графика без нарушения авторских прав.  
- Разработка выполняется одним человеком.

---

## Запуск
1. Скачайте или клонируйте репозиторий.  
2. Откройте `index.html` в браузере.  
3. Начинайте обучение.
