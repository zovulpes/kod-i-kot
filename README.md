# О проекте
В "**Код и Кот**" вы можете выполнять задания по процедурному программированию на **python**. На данный момент реализовано 6 уровней по темам: ввод и вывод данных, арифметика, условия цикл for и while.

Реализовано продолжение сохраненного прогресса с помощью LocalStorage - система вернет вас к уровню, на котором вы остановились.

Первое, что бросается в глаза при открытии "Код и кот " - милый дизайн и не менее милый пиксельный кот, который даёт подсказки при выполнении уровней.
