Уровень 6: Цикл while

Аннотация. Урок посвящен циклу while. Мы научимся создавать программы, которые повторяют определенные действия, пока выполняется некоторое условие.

Мы уже рассмотрели работу цикла for, который является счетным — он выполняется определённое количество раз. Однако не все задачи можно решить заранее зная число итераций. В таких ситуациях нам нужен цикл while.

Структура

while условие:
    блок кода

Сначала проверяется условие. Если оно истинно (True), выполняется тело цикла — блок кода с отступом. Затем снова проверяется условие и при истинности снова выполняется тело цикла.

Инициализация и изменение переменных

Важно корректно инициализировать переменные, используемые в условии, и изменять их внутри цикла, иначе можно получить бесконечный цикл. 

i = 0
while i < 5:
    print(i)
    i += 1

В примере выше переменная i увеличивается на 1 в каждом проходе, поэтому цикл завершится.

Считывание до стоп-значения

Один из распространённых шаблонов — считывание данных до специального слова-«стопа». 

text = input()
while text != 'stop':
    total += int(text)
    text = input()

Шаблон используется, когда количество вводимых строк заранее неизвестно.

Цикл while vs цикл for

Цикл for удобен, когда известно число итераций; while — когда число итераций зависит от входных данных. Иногда одну конструкцию можно заменить другой, но не всегда. 

# Пример: вывод чисел, кратных 33, с использованием while
i = 0
while i < 100:
    if i % 33 == 0 and i != 0:
    print(i)
    i += 1

Избегаем бесконечных циклов

В большинстве задач следует избегать бесконечных циклов. Цикл должен содержать изменение состояния (например, увеличение счётчика или чтение нового значения), которое в конечном счёте сделает условие ложным.

Логические выражения в условии

Условие в цикле while может содержать логические операторы and, or, not, сравнения, и даже вызовы функций. 

while 3 <= age <= 6:
print('Вы ребёнок')


while not finished and attempts < 5:
    attempts += 1

Примечание 1. Цикл while получил своё название из-за характеристики "пока" — блок выполняется, пока условие истинно.

Примечание 2. Цикл while называют циклом с предусловием — условие проверяется перед первой итерацией.

Примечание 3. Цикл while может не выполниться ни одного раза, если условие изначально ложно.

Примечание 4. Условие в цикле while может содержать логические операторы or, and, not.

Задание: До КОНЦА

На вход программе подаётся последовательность слов, каждое слово на отдельной строке. Концом последовательности является слово «КОНЕЦ». Само слово «КОНЕЦ» не входит в последовательность. Выведите члены этой последовательности (каждое слово на отдельной строке).
